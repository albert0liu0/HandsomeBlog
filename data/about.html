About This Site

Introduction

本網站為 Handsome Liu （劉瀚聲）的個人網站的前端部份，而後端部份使用 node.js ，放置於 https://github.com/albert0liu0/HandsomeBlog 中，目前尚未與前端連接。

Why I Don't Use Others' Blog System

每個人對於 blog system 的追求不盡相同。我認為一個相對於我而言是完美的 blog system ，應該具備且僅具備我想要的功能，而這不是沿用別人的 blog system 所能達到的。此外，他人的 coding style 會造成閱讀及理解上的不易，從而導致維護上的困難。

Why I Don't Use Others' Front-end Templates

我不使用前端模板的理由類似上面所述。美感因人而異， coding style 更是如此。

Why I Seldom Use Others' Framework

在一定程度內，我希望我有能力自己實作一件事情後，再考慮使用其他的框架來輔助實作。這不僅僅是為了個人能力的鍛鍊，同時也包含了對程式效率上的追求，以及避免抽象漏洞。
